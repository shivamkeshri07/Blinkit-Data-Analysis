# 🛒 Blinkit Data Analysis | Power BI Dashboard

## 📌 Overview

This project analyzes Blinkit's sales, outlet performance, product categories, and customer ratings using Power BI. The dashboard provides interactive insights that help understand business performance and identify growth opportunities.

## 🎯 Objectives

- Analyze total sales performance
- Track key KPIs
- Compare outlet types and locations
- Identify Top Selling product categories
- Evaluate customer ratings
- Generate business insights through visualization

## 📊 Key Metrics

| KPI | Value |
|-----|-------|
| 💰 Total Sales | $1.20M |
| 📦 No. of Items | 8,523 |
| 📈 Avg Sales | $141 |
| ⭐ Avg Rating | 3.9 / 5 |
